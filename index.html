CSS stands for Cascading Style Sheets. It is a styling language and it is called "cascading" because CSS styles can overlap one another.

Types of Style Sheets

  in <head> of html add <link rel="stylesheet" href="path_of_styles.css">
  in styles.css file:
  selector{
  property:value //style definiton while using external CSS
  }
  <style> css code here </style> //when using internal CSS
  <tag style= "property:value;"> //when using inline CSS

Colors

Types of colors:

  1. named colors: limited and are written directly, no shades,
    background-color:black;
  2. RGB: composition of red green blue, can generate 16 million colors. syntax: rgb(red,green,blue). Eg:rgb(0,0,0)=black, rgb(255,255,0)=yellow,
    background-color:rgb(255,102,102);
  3. hexadecimal values: should be 6 char long with prefix # Eg: #ffffff=white, #ff0000=red,
    background-color:#79bac1;

Font Styles

  • font-family :the font type, best to have more than one
  • font-size: can be given in px or small, x-small, xx-small, large, etc
  • font-weight :for bolding, can be bold/bolder/lighter/normal/px, default value=normal
  • font-style :for italicizing or making it oblique, font-style:italic/normal/oblique
  • font-stretch : to make the text narrower or wider, can be normal, expanded, condensed, semi/extra/ultra expanded
  • color : to change the font color of any text.
  • font:a shorthand property as
    font: style  variant(like small-caps)  weight  size/line-height  family;

Text Styles

  • letter-spacing: px;
  • word-spacing: px; used for introducing space between words.
  • line-height:px; for height between lines.
  • text-decoration: underline/overline/linethrough;
    • text-decoration-color
    • text-decoration which can be solid/wavy/dotted/dashed/double 
      text-decoration-style: double
  • text-align can be left/right/center/justify
    text-align:center;
//default value:left
  • text-indent: specifies left margin for first line in paragraph. This is rarely used.

CSS Selectors:

Tag selector 
 tag{ property: value; }
ID selector (must be distinct) 
 #idname{ property:value; }
Class selector (for a group of elements) 
 .classname{ property:value; }
Compound selector (for a specified tag having a specific class) 
 tag.class{ property:value; }
Grouping selector (selects the specified group of tags) 
 tag1,tag2,tag3{ property:value; }
Child selector (selects all the child tags of specified parent tag, can be grandchildren too) 
 parenttag childtag{ property:value; }
Attribute selector (selects tags having specified attribute) 
 tag[property1:value1]{ property2:value2; }
Hover selector (applies style only when the mouse pointer is placed over the element) 
 tag:hover{ property:value; }
Focus selector (style applied only when the focus i.e. cursor is inside an element like a textbox) 
 tag:focus{ property:value; }
Universal selector (selects all the tags in the web page including html, head, body, etc) 
 *{ property:value; }

CSS Style Precedance

Higher priority overrides the same property's value of the lower priority

  1. ID Selector
  2. Attribute Selector
  3. Class Selector
  4. Child Selector
  5. Direct Child Selector
  6. Tag Selector
  7. Browser default style

Transitions

It is a process of changing a CSS property's value gradually, based on the specified no. of seconds

It supports only pixels based color based properties like width, height, opacity, font-size, border-width, background-color, color, border-color etc.

  selector
  {
    property: startvalue;
    transition: property seconds;
  }
  selector:hover
  {
    property: endvalue;
  }

Transformations

It is used to display the element in a different visual dimension.

Types of transformations

transform-origin:specifies the fixed point for rotate and scale transformations

transform-origin: top left/top center/top right/ center left/ center center/ center right/bottom left/bottom center/bottom right;
  Eg: transform-right:top center;
  1. Rotate (used to rotate an element based on the specified degrees) 
          transform: rotate(deg);
      Eg- transform: rotate(45deg);
  2. Skew (wrists the element) 
          transform: skew(deg);
  3. Scale (shows the element in large size or small size visually) 
          transform: scale(number);
  4. Translate (changes the visual position of the element) 
          transform: translate(x,y);
      Eg- transform: translate(10px,10px);
  5. Multiple Transformations (more than one transformation at a time) 
          transform: rotate(deg) scale(num) translate(x,y) skew(deg);
      Eg- transform: rotate(90deg) scale(3) translate(25px,25px) skew(15deg);

Animations

They are a group of transitions which are performed one after the other over a period of time.

A transition contains only two points(starting pt and ending pt) but an animation contains multiple points of milestones.

  selector{
   animation: animationname seconds(s);
  }
  @keyframes animationname{
   0%{ property: value; property: value; }
   25%{ property: value; property: value; }
   50%{ property: value; property: value; }
   75%{ property: value; property: value; }
   100%{ property: value; property: value; }
  }

  Eg: .divname{
  background-color:black;
  font-size:40px;
  width:200px; height:200px;
  }
  .divname:hover{
  cursor:pointer;
  animation: myanimation 10s;
  }
  @keyframes myanimation{
   0%{ transform: translate(0px,0px); }
   25%{ transform: translate(50px,0px); }
   50%{ transform: translate(50px,100px); }
   75%{ transform: translate(0px,100px); }
   100%{ transform: translate(0px,0px); }
  }
